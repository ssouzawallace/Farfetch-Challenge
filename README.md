
# Farfetch-Challenge 

An iOS application to discover games with the most views on Twitch with to possibility to save them locally.


# Main app

The app consist of three sections where the user can see the list of games ordered by number of views, the list of games saved as favorites and a details screen to see the game cover art with higher resolution.
